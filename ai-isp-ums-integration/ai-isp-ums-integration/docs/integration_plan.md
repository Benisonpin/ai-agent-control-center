# AI ISP + UMS 整合計畫

## 專案概述
整合 UMS (Unified Memory Subsystem) 到現有的 AI ISP 系統。

## 目標
- 實現零拷貝記憶體共享
- 提升系統效能 40%+
- 降低功耗 35%
