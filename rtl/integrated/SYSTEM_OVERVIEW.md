## Key Features

1. **HDR Processing**
   - 3-exposure fusion (HCG/LCG/VS)
   - DCG dual conversion gain
   - AI-adaptive tone mapping

2. **Complete ISP Pipeline**
   - 12-bit to 24-bit RGB processing
   - Full image enhancement pipeline
   - Real-time 4K support

3. **AI Enhancement**
   - 8 scene types detection
   - Dynamic parameter optimization
   - Quality assessment

4. **Memory Management**
   - UMS frame buffer integration
   - Double buffering
   - Zero-copy optimization

5. **Configuration**
   - AXI4-Lite interface
   - Runtime parameter adjustment
   - Debug and monitoring
